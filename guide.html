A script and web interface for mirroring XML of the PubMed Central Open Access Subset Code: http://github.com/mchelen/pmc.oas.xml.mirror Based on version 0.1: http://github.com/mchelen/pmc.oas.xml.mirror/tree/0.1 mirror.sh Downloads and extracts current copy of PubMed Open Access Subset XML archives. Loop through and download each archive with wget then extract to a specified directory. A faster download client, aria2c, can be used for multiple concurrent connections. index.php Web interface to provide XML file of requested PubMed Central ID. We search the filesystem with user input PMC ID Display the file as XML form.html Form for user input of PubMed Central ID. Passed as GET variable "pmcid" http://pmcxml.opensci.info/?pmcid=123456 Browse: http://pub.opensci.info/pmc.oas.xml.mirror/files/
